Markdown 언어 소개

1. Markdown Language 소개

Markdown은 2004년에 존 그루버(John Gruber)와 아론 스워츠(Aaron Swartz)가 개발한 경량 마크업 언어입니다. HTML보다 간단하고 쉽게 작성할 수 있으며, 웹에서 보다 쉽게 문서를 작성하고 서식을 지정하는 데 사용됩니다. Markdown의 주요 장점과 특징은 다음과 같습니다:

장점:

  • 간단하고 가독성이 높은 문법: 일반 텍스트에 태그를 사용하지 않고도 구조적인 문서를 작성할 수 있습니다.
  • HTML로 변환이 가능: Markdown으로 작성한 문서를 HTML로 변환하여 웹에서 사용할 수 있습니다.
  • 여러 플랫폼에서 지원: 다양한 에디터와 플랫폼에서 Markdown을 지원하여, 텍스트 문서의 호환성을 보장합니다.
  • 버전 관리 용이성: Git 등 버전 관리 시스템과의 호환성이 높아, 협업 문서 작업에 용이합니다.

2. Markdown Editor 소개: Dillinger.io

Dillinger.io는 인기있는 Markdown 편집기 중 하나입니다. 주요 기능과 특징은 다음과 같습니다:

주요 기능:

  • 실시간 미리보기: Markdown으로 작성된 내용을 실시간으로 HTML, Styled HTML, Markdown, PDF 형태로 미리 볼 수 있습니다.
  • 클라우드 연동: Dropbox, Bitbucket, Github, Medium, Google Drive, One Drive와 같은 클라우드 서비스와 연동하여 작업한 내용을 자동으로 동기화할 수 있습니다.
  • 다양한 파일 출력 형식: Markdown으로 작성된 내용을 HTML, Styled HTML, Markdown, PDF 파일로 저장할 수 있습니다.
  • Save, Import 지원: 지정한 클라우드 서비스에 저장하고, 불러오는 기능을 제공하여 편리한 작업을 가능하게 합니다.

Dillinger.io는 사용하기 쉽고 다양한 기능을 제공하여 Markdown 작성을 편리하게 지원합니다.

3. Markdown Language 예제 (추가 예정)

아래는 간단한 Markdown 문법을 이용한 예제입니다:

# 제목 (h1)
## 소제목 (h2)

이런 식으로 h1 ~ h6까지 있습니다.

이것을 이용해
문서 내 [TOC] 또는 {toc}를 입력할 경우
#(해시태그) 기준으로 목차가 자동으로 생성됩니다.

TOC는 Table of Contents의 약자입니다.

따라서 #(해시태그) 구조를 잘 구성하는 것이 
구조적인 글쓰기와 목차 구성에 큰 도움이 됩니다.

**굵은 글씨**와 *기울임체*도 표현 가능합니다.

- 리스트
- 목록

1. 번호
2. 리스트

[링크](http://www.example.com)

![이미지](http://www.example.com/image.jpg)

이와 같이 Markdown은 간단하면서도 다양한 텍스트 서식을 지원하여, 다양한 용도로 사용됩니다.

Markdown은 쉽고 빠르게 문서를 작성하고, 다양한 플랫폼에서 호환성을 유지할 수 있는 강력한 도구입니다.
